# Документация NewsHub

Эта папка содержит всю документацию проекта NewsHub.

## Структура документации

### Основные файлы

- **README.md** (в корне проекта) - Главный файл документации с описанием проекта, установкой и использованием
- **API.md** - Подробное описание REST API endpoints с примерами
- **er-diagram.txt** - Текстовая ER-диаграмма базы данных
- **architecture.txt** - Текстовая архитектурная схема системы

### Папки

- **screenshots/** - Скриншоты интерфейса приложения
  - `home.png` - Главная страница
  - `register.png` - Регистрация
  - `login.png` - Вход
  - `favorites.png` - Избранное

## Визуальные диаграммы

1. **ER-диаграмма** (`er-diagram.png`) - Схема базы данных
2. **Архитектурная схема** (`architecture.png`) - Схема взаимодействия компонентов

## Обновление документации

При изменении проекта обновите:

1. **README.md** - если изменились функции, установка или структура
2. **API.md** - если изменились API endpoints
3. **er-diagram.txt** - если изменились модели данных
4. **architecture.txt** - если изменилась архитектура
5. **Скриншоты** - если изменился интерфейс
